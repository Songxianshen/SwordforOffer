## 最长不含重复字符的子字符串
> 题目:
 
&emsp;&emsp;请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。假设字符串中只包含了
`'a'~'z'`的字符。例如，在字符串`"arabcacfr"`中，最长的不含重复字符的子字符串是`"acfr"`，长度为4。
> 解题思路： 

* 利用动态规划
1. 首先定义函数`f(i)`表示第i个字符为结尾的不包含重复字符的字符串的最长长度。我们从左到右逐一扫描字符串中每个字符。当我们计算
`f(i)`时，我们已经知道`f(i-1)`了。
